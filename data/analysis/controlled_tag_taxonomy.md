# Controlled Tag Taxonomy

This is a first-pass controlled tag set for grouping Ehrman Blog posts. The goal is not to preserve every legacy WordPress tag, but to create a smaller, consistent vocabulary that can support browsing, clustering, and later editorial review.

## Recommended Model

Use multiple tags per post:

- `content_area`: 1-3 tags for the main subject.
- `theme`: 0-4 tags for recurring theological, historical, or social themes.
- `method`: 0-3 tags for scholarly method or source type.
- `format`: 0-2 tags for the kind of post.

| Tag | Label | Use When |
| --- | --- | --- |
| `historical-jesus` | Historical Jesus | Jesus as a historical figure, his life, sayings, death, messianic identity, or historicity debates. |

| `paul-and-letters` | Paul and Letters | Paul, Pauline theology, disputed Pauline letters, other epistles, apostolic correspondence. |
| `acts-and-revelation` | Acts and Revelation | Acts of the Apostles, Revelation, apocalyptic imagery, early mission narratives. |
| `hebrew-bible-judaism` | Hebrew Bible and Judaism | Hebrew Bible, Second Temple Judaism, Jewish law, Jewish groups, Jewish-Christian relations. |
| `early-christianity` | Early Christianity | Christian origins, first centuries CE, church fathers, early Christian communities and practices. |
| `christian-apocrypha` | Christian Apocrypha | Non-canonical gospels, apocryphal acts, infancy gospels, lost scriptures. |
| `greco-roman-world` | Greco-Roman World | Roman Empire, pagan religions, mystery religions, imperial context, broader ancient Mediterranean culture. |
| `church-history` | Church History | Later historical developments, Constantine, councils, Reformation, Christianity after the earliest period. |

## Notes for Application

- Avoid using broad tags only because a common word appears. For example, `historical-jesus` should not apply to every post that merely mentions Jesus.
- Prefer 3-7 controlled tags per substantive post.
- Short announcement/video posts can receive only format tags such as `blog-community` and `media-video`.
- Legacy categories should be retained separately as source metadata, not treated as the final controlled taxonomy.
